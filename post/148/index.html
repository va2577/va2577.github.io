WSL の Ubuntu に Jupyter Notebook をインストールしました

April 28, 2018
広告

Jupyter Notebook については、以前、記事を書いたことがあって、そのときに使いました。 今回は、 WSL の Ubuntu でも使えるようにインストールしてみました。

環境

  • Ubuntu 16.04 LTS
  • Python 3.5.2
  • jupyter-1.0.0

pip

WSL の Ubuntu の Python に pip がないので、 Python の仮想環境の中に pip をインストールするところから始めました。

$ python3 -m venv --without-pip jup
$ cd jup
$ ls
bin  include  lib  lib64  pyvenv.cfg
$ source bin/activate
(jup) $

仮想環境の jup を作成して、有効にしました。

pip をインストールするのに必要なファイルをダウンロードします。

(jup) $  curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

If you have Python 3 installed (which is recommended):

python3 -m pip install jupyter

Running the Notebook

Basic Steps

  1. Start the notebook server from the command line:

     jupyter notebook

  2. You should see the notebook open in your browser.
